## 📽️ OverTheCam (캠 건너 불구경)
(우리 로고 이미지지)


재미에 의한, 재미를 위한.

실시간으로 논쟁이 필요한 사람들을 위한 화상 논쟁 플랫폼


OverTheCam은 일상의 모든 논쟁을 재미있게 해결하는 플랫폼입니다.
친구랑 의견이 다를 때, 동료와 입장이 갈릴 때,
실시간 화상으로 배틀하고 투표와 배팅으로 승자를 가려보세요!
