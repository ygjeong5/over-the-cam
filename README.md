# **🚀 Tech Stack**  

### **Frontend**  
- React.js | JavaScript (Vanilla 포함)  
- Socket.io Client | Axios  

### **Backend**  
- Java 17 | Spring Boot 3.2.3  
- Spring Data JPA | Spring WebSocket (STOMP) | Spring Cloud AWS
